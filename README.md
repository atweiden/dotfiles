# Dotfiles for Arch

## Installation

```bash
git clone https://github.com/atweiden/dotfiles
cd dotfiles
chmod u+x bootstrap.sh
./bootstrap.sh
```

## Usage

### Vim

```bash
vim
:BundleInstall
:Matrix
```

### Git, Mercurial

```bash
vim $HOME/.gitconfig $HOME/.hgrc
```
